# Mining Operations Management System

MinERP is a Mining Operations Management System -- Multi-tenant ERP for payroll, personnel, financial tracking, and operational compliance in Peruvian mining operations. Web application mining operations management system. OpenMines is a Python-based simulation environment designed for truck dispatching in mining operations. ERPNext Cargo Management is a fully-fledged ERPNext solution designed for freight forwarding companies. AuMMS ( Gold Manufacturing Management System ) is a Frappe App to facilitate the Operations in Gold Manufacturing. Triton Mining Co. is an open-source deep-sea exploration initiative focused on AI-driven AUVs, seabed mapping, and environmental monitoring. Python library for mineral prospectivity mapping. Apache Geode is a data management platform that provides real-time, consistent access to data-intensive applications throughout widely distributed cloud architectures. Python module for drillhole database management. HyperSpy is an open source Python library for the interactive analysis of multidimensional datasets. Mapbox GL JS is a JavaScript library for interactive, customizable vector maps on the web. OpenJUMP Desktop-GIS software provides a swiss-knife of tools to analyze and edit huge GIS-datasets.Reactor Bill Of Materials (tracking reactor-core, reactor-netty and more). The Geostatistics & Machine Learning C++ Library. Machine learning framework for reservoir simulation. Spatial and spatio-temporal geostatistical modelling, prediction and simulation. Albion plugin provides tools to create 3D geological models in QGIS. A Blender add-on with a collection of tools for 3D geological modelling, primarily designed for mineral exploration. FreeCAD Trails Workbench (Transportation and Geomatics). Open-source python module for mineral resource estimation and geostatistics. A PyQt5 app catered to the exploration industry for visualizing geologic drill hole data.

The directory structure includes a specific folder for each phase of development and engineering programs:

- __MBSE__: The root folder Arcadia (ARChitecture Analysis and Design Integrated Approach) is a method of tools dedicated to systems engineering and architecture, supported by the Capella modeling tool.    
  - __CAD__: For computer-aided design.
  - __CAM__: Dedicated to the part manufacturing and assembly process.
  - __CAS__: For simulation of end-to-end functionality and performance analysis.
